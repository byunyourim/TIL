### 메서드
- push() : 스택의 가장 윗 부분에 삽입
- pop() : 스택의 가장 윗 부분 제거
- peek() : 스택의 가장 윗 부분 반환
- isEmpty() : 비어있는 겨우 true 반환



### 사례
- 웹 브라우저 방문의 뒤로가기 기능, 실행 취소 
- 역순으로 문자열 만들기, 수식의 괄호 검사
- 깊이 우선 탐색, 백트래킹, 재귀 함수
