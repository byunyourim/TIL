### 재정의
|10진법	|124 나라	|10진법	|124 나라|
|------|------|--------|-------|
|1 |	1	| 6	| 14|
|2	|2	|7	|21|
|3	|4	|8	|22|
|4	|11	|9	|24|
|5	|12	|10	|41|



3(1) : 1      2      4     
6(2) : 11     12     14
9(3) : 21     22     24
12(4): 41     42     44

3진법...

나머지를 더한다! 문자열로
1. n을 3으로 나눈다
2. 6인 경우 몫은 2, 나머지는 0 -----> 나누어 떨어지는 경우 몫-1
2-1. 14인 경우 몫은 4, 나머지 2  
     몫이 3이상인 경우 또 3으로 나눈다(반복)
     -> 몫은 1, 나머지 1
3. 1은 1로, 2는 2, 0은 4로 변환하여 수를 더한다 (112)

### 알고리즘
O(logN)   
반복문 while (n>0) 사용  


n을 3으로 나눌 때 필요한 횟수, 반복문이 실행되는 횟수만틈 시간이 소요된다.  



### 의사코드
```
answer

while(n > 0) {

나머지 = n % 3;

  if 나머지가 0 이라면
      answer에 4 추가;
      몫을 3으로 나눈 후 -1
  else
      answer에 1 또는 2 추가;
      몫을 3으로 나눔;
}

answer을 역순으로 반환

```
