## 재정의
- sequence = [1, 2, 3, 4, 5]	
- sum = 7	
- index = [2, 3]

연속된 수의 값이 sum이 될때 인덱스를 구하기!!  
여러개인 경우 가장 앞에 것으로!!!

## 알고리즘
for 문
  
- O(N)




## 의사코드
```
for sequence를 돌며 값들을 더한다
  if : 더한 값 >  sum ==> 맨 앞의 값 제거
  if : 더한 값 < sum ==> 값 추가ㅏ
  if : 더한 값 == sum ==> 처음 인덱스와, 마지막 인덱스 리턴 
```
